Cómo Agregar una Lista Ordenada en HTML

En este tutorial, aprenderás a crear una lista ordenada en tu documento HTML utilizando la etiqueta <ol>

Las listas ordenadas son útiles cuando necesitas mostrar elementos en un orden específico. Aquí tienes un ejemplo básico:

      
<ol>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
  <li>Tercer elemento</li>
</ol>

Esto generará una lista numerada y mostrará los elementos en el orden indicado.

Cómo Agregar un Enlace en HTML

En este tutorial, aprenderás a insertar un enlace a otra página web en tu documento HTML.

Para agregar un enlace, utiliza la etiqueta <a> seguida de la propiedad href que indica la URL del destino. Aquí hay un ejemplo:

        
<a href="https://www.ejemplo.com">Visitar Ejemplo.com</a>

En este caso, al hacer clic en el enlace, llevará al usuario a la página web Ejemplo.com.

Cómo Agregar un Formulario en HTML

En este tutorial, aprenderás a crear un formulario básico en tu documento HTML utilizando las etiquetas <form><input><button>

Aquí hay un ejemplo sencillo de un formulario de contacto:

      
<form action="/enviar" method="post">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" required>

  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <label for="mensaje">Mensaje:</label>
  <textarea id="mensaje" name="mensaje" required></textarea>

  <button type="submit">Enviar</button>
</form>

Este formulario simple solicita al usuario que ingrese su nombre, correo electrónico y un mensaje. Al hacer clic en el botón "Enviar", se enviará el formulario a la ruta "/enviar" utilizando el método POST.

Cómo Agregar un Vídeo en HTML

En este tutorial, aprenderás a incorporar un vídeo en tu documento HTML utilizando la etiqueta <video>

Utiliza la siguiente estructura básica para agregar un vídeo:

      
<video width="640" height="360" controls>
  <source src="ruta-del-video.mp4" type="video/mp4">
  Tu navegador no soporta el elemento de video.
</video>

Sustituye "ruta-del-video.mp4" con la URL o ruta local de tu archivo de vídeo. La etiqueta <source> permite especificar diferentes formatos de vídeo para garantizar la compatibilidad en varios navegadores.

El atributo controls agrega controles de reproducción al reproductor de vídeo.

Cómo Incrustar un Mapa de Google en HTML

En este tutorial, aprenderás a añadir un mapa de Google a tu página HTML utilizando la etiqueta <iframe>

Aquí tienes un ejemplo básico:

      
<iframe
  width="600"
  height="450"
  frameborder="0"
  style="border:0"
  src="https://www.google.com/maps/embed/v1/place?q=nombre+del+sitio&key=TU_CLAVE_API"
  allowfullscreen>
</iframe>

Sustituye "nombre+del+sitio" con la ubicación o nombre del lugar que deseas mostrar en el mapa. Además, asegúrate de reemplazar "TU_CLAVE_API" con tu propia clave de API de Google Maps.

Cómo Crear una Tabla en HTML

En este tutorial, aprenderás a construir una tabla básica en tu documento HTML utilizando las etiquetas <table> <tr> (fila), <th> (encabezado) y <td> (celda).

Aquí tienes un ejemplo sencillo:

      
<table border="1">
  <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
  </tr>
  <tr>
    <td>Celda 1-1</td>
    <td>Celda 1-2</td>
  </tr>
  <tr>
    <td>Celda 2-1</td>
    <td>Celda 2-2</td>
  </tr>
</table>

Esta estructura crea una tabla con dos filas y dos columnas. Puedes personalizar el contenido y el número de filas y columnas según tus necesidades.

Cómo Dibujar en un Canvas HTML

En este tutorial, aprenderás a utilizar la etiqueta <canvas> para dibujar gráficos en tu documento HTML mediante JavaScript.

Aquí hay un ejemplo básico:

      
<canvas id="miCanvas" width="300" height="200"></canvas>
<script>
  var canvas = document.getElementById('miCanvas');
  var contexto = canvas.getContext('2d');

  contexto.fillStyle = 'blue';
  contexto.fillRect(50, 50, 200, 100);

  contexto.strokeStyle = 'red';
  contexto.lineWidth = 5;
  contexto.strokeRect(50, 50, 200, 100);

  contexto.beginPath();
  contexto.arc(200, 150, 30, 0, 2 * Math.PI);
  contexto.fillStyle = 'green';
  contexto.fill();
</script>

Este código crea un lienzo (canvas) y utiliza el contexto 2D para dibujar un rectángulo azul y un círculo verde.
